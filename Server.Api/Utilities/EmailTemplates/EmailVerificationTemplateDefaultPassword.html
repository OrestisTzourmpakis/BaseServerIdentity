Καλωσορίσατε στο πρόγραμμα επιβράβευσης επισκεπτών - πελατών

Παρακάτω θα βρείτε τους κωδικούς σας για εγγραφή στην πλατφόρμα επιβράβευσης πελατών (loyalty), στο πλαίσιο υλοποίησης του έργου ανοικτό κέντρο εμπορίου δήμου Λεβαδέων, στο οποίο συμμετέχει ως συν δικαιούχος το επιμελητήριο Βοιωτίας.

Παρακαλούμε όπως επικυρώσετε το email σας:

Επικύρωση email

Είσοδος στην πλατφόρμα: ΕΙΣΟΔΟΣ

Username: useremail

Παρακάτω θα βρείτε τον κωδικό πρόσβασης: defaultPassword

Εάν επιθυμείτε να αλλάξετε τον κωδικό, πατήστε "Reset Password" κατά την είσοδο σας στην πλατφόρμα.
